# Sanitary Product Images Setup

To add images from your JustDial gallery to the sanitary tab:

## Steps to Download Images:

1. Visit: https://www.justdial.com/Bangalore/Swastik-Ceramics-Co-Near-Raghavendra-Swamy-Temple-Jayanagar-5Th-Block/080PXX80-XX80-260310115001-V1B1_BZDET/gallery?tab=all&menu_slider=0

2. Right-click on each image in the gallery and select "Save image as..."

3. Save the images to: `src/assets/sanitary/`
   - Name them as: `sanitary-1.jpg`, `sanitary-2.jpg`, etc.

4. The app will automatically display these images in the Sanitary tab.
