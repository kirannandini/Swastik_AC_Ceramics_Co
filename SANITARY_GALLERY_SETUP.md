### Step 2: Download Images
1. Right-click on each image in the gallery
2. Select "Save image as..." or "Download image"
3. Save images to: `src/assets/sanitary/`
4. Name them exactly as: `gallery-1.jpg`, `gallery-2.jpg`, `gallery-3.jpg`, etc.

### Step 3: Image Mapping
The app expects these specific filenames:
- `gallery-1.jpg` → Premium Sanitaryware Collection
- `gallery-2.jpg` → Designer Bathroom Fittings
- `gallery-3.jpg` → Luxury Wash Basins

- `gallery-7.jpg` → Water Closets
- `gallery-8.jpg` → Bath Tubs
- `gallery-9.jpg` → Kitchen Sinks
- `gallery-10.jpg` → Faucets & Taps

### Step 4: View Results
- Navigate to the Products page
- Click on the "Sanitary" tab
- All your downloaded images will be displayed in a grid

- If images don't appear, check that filenames match exactly
- Clear browser cache and refresh the page
- Ensure images are saved in `src/assets/sanitary/` folder
